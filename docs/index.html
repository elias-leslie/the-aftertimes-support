Run on Linux

  1. Download the Linux review zip.
  2. Optional hash check: sha256sum the-aftertimes-realm1-linux-a5392e7c.zip should print 4d25cd74e4212de86dd64c7e3fa9b4144d267b31d47192af3def51c276c0904e.
  3. Unzip it, keep the-aftertimes.x86_64 and the-aftertimes.pck in the same folder, then run:
    chmod +x the-aftertimes.x86_64 && ./the-aftertimes.x86_64

Privacy / telemetry

Review builds may write local-only JSONL playtest events for player deaths, room builds, and reactor completions. No network analytics are sent. Open the telemetry review guide.

Do not post secrets, private account information, tax/payment details, unreleased credentials, or unredacted local logs/paths.
