Run on Linux

  1. Download the Linux review zip.
  2. Optional hash check: sha256sum the-aftertimes-realm1-linux-28b02d4d.zip should print 5edeb25c0f80cd2a8a84fd061c790acadf31a9da04c86cb4b5a337bdfa5b4e33.
  3. Unzip it, keep the-aftertimes.x86_64 and the-aftertimes.pck in the same folder, then run:
    chmod +x the-aftertimes.x86_64 && ./the-aftertimes.x86_64

Current review status

The review build is available, but paid launch is still blocked until these public verdicts are recorded or explicitly deferred:

  • External QA: needs a fresh unaided completion playtest verdict through the External QA playtest report.
  • Art / animation / audio: needs running-game plus artifact review through the Art / animation / audio review.
  • Legal / store / business: needs store/legal/platform/pricing approval, or accepted deferrals, through the Legal / store / business review.

Privacy / telemetry

Review builds may write local-only JSONL playtest events for player deaths, room builds, and reactor completions. No network analytics are sent. Open the telemetry review guide.

Do not post secrets, private account information, tax/payment details, unreleased credentials, or unredacted local logs/paths.
